# Typing/Rythym Game

This python progam is rhythm/typing game, designed as part of year 12 major works project. The player creates an account that is saved in an encrypted file. The player selects which game they want to play, rhythm or typing, allowing them to figure out how fast they type or reaction time. Players has 3 songs to choose from in rhythm game with unique falling tiles, while typeosarus has different amount of randomised words they have to type.

## Features
- Multiple selection of amount of words

- Song selection: Prairies, Suzume, White Keys

- Results page with accuracy, time, wpm

- Results page with amount of points, accuracy, and ranking

- Particle affects within rhythm game

- 8 keys to press to hit notes (a, s, d, f, h, j, k, l)

- Wordbank with over 300 000 words


## Requirements
To run this program, you need to install the following dependencies:

Pygame (used for gui display and particle affects)

Cryptography (used to encrypt passwords in login system)

Install dependencies
``` bash
pip install -r requirements.txt

```
